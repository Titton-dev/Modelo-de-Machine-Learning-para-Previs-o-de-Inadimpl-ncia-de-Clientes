# Modelo-de-Machine-Learning-para-Previs-o-de-Inadimpl-ncia-de-Clientes
Desenvolvimento de um modelo de Machine Learning em Python para prever risco de inadimplência de clientes, utilizando técnicas de classificação, pré-processamento de dados categóricos e avaliação de desempenho. Projeto aplicado com foco em análise de dados e tomada de decisão.
